## Локальный запуск

### 1. Клонировать проект

```bash
git clone https://github.com/vellun/next-space
```

### 2. Перейти в директорию проекта

```bash
cd next-space
```

### 3. Установить зависимости

```bash
yarn install
```

### 4. Запустить

- dev режим

```bash
yarn dev
```

- prod режим

```bash
yarn build
```

```bash
yarn start
```

По умолчанию приложение будет доступно на порту 3000

```bash
http://localhost:3000
```
